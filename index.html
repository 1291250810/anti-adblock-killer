Why it does not work

  1. Check if userscript manager is activated. (Greasemonkey, Tampermonkey, NinjaKit, Violentmonkey)
  2. Check if Anti AdBlock Killer is activated.
  3. Check if you have installed the lastest version of Anti AdBlock Killer | Userscript
  4. Check if you have subscribe to Anti AdBlock Killer | Filters for Adblockers - Subscribe

  5. Check if you have another anti adblock user script installed (Ex: Anti-AntiBlock Plus, Anti-Adblocker), please disable or remove it, because it can generate the conflict.
  6. Check if the site is in "Supported Sites".
  • If the site is not in it, please report it here.
  • If it exists, but the script does not work, it probably means that the anti adblock protection has changed in it please report it here
